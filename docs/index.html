About TabTalkAI

TabTalkAI is a Chrome extension that lets you chat with any webpage using conversational AI. Instantly generate summaries, key points, analysis reports, and get answers to FAQs — all powered by your own free Gemini API key.

Features

  • 📝 Generate summaries of webpage content
  • 🔑 Extract key points instantly
  • 📊 Create analysis reports
  • ❓ Get answers to frequently asked questions (FAQs)
  • 🔒 Uses your own free Gemini API key for privacy and control
  • 🌙 Supports dark and light modes

Installation

  1. Get the Extension:
    • Download from GitHub
  2. Obtain a Gemini API Key:
    1. Visit ai.google.com/studio
    2. Create an API key & enable Gemini API
    3. Paste the key into TabTalk AI
  3. Setup:
    • Open TabTalkAI in Chrome
    • Go to settings and enter your Gemini API key
